Cada box possui 4 edges: margin edge, border edge, padding edge e content edge.
Content: é onde está o conteudo da BOX.
Padding: é a área de preenchimneto da BOX
Border: É a borda da BOX
Margin: estende a área de borda com um espaço vazio.
ATENÇÃO: A área de margin não é somada entre os elementos vizinhos.
No caso, a maior área prevalece.
Na hora de configurar espaçamento de alguma área, a ordenação deve estar sentido horário
Exemplo: margin:15px 10px 15px 25px
A ordem seria: TOPO DIREITA BAIXO ESQUERDA
Se houver apenas um valor, se aplica para todos os lados.
