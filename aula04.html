Tags que podem vir dentro dos parágrafos:
- 'b' : Negrito
- 'strong' : Deixar uma palavra mais 'forte' (parecido com negrito)
::::: A diferença entre os dois é: 'strong' é um estado lógico, e 'b' é um estado físico.
- 'i ' : Itálico
- 'em' : Emphasis (parecido com itálico)
::::: 'EM' da ênfase, itálico é um estado físico.
- 'a href=""' : LINK

Tag 'img': Adiciona imagem:

- 'del ' : "DELETA" uma palavra / linha: EXEMPLO DO DEL
- 's ' : Serve para riscar uma palavraEXEMPLO DO S
- 'ins ' : Serve para inserir palavra / linha EXEMPLO DO INS
- 'u ' : Serve para sublinhar uma palavraEXEMPLO DO U
----- > As diferenças entre 'del' e 's' / 'ins' e 'u', seguem a mesma linha do 'b' e 'strong'
- 'small' : Diminui o texto
- 'sup' : Colocar um numero elevado. EXEMPLO: 22
- 'sub' : Colocar um numero rebaixado. EXEMPLO: H2O
- 'q ' : Citação. Exemplo: Texto de citação
- 'br ' : quebra de linha
- TODAS AS TAGS ACIMA PODEM SER APLICADAS DENTRO DA TAG DE PARÁGRAFO ('p /p'), ou seja, INLINES

Por fim, as tags SPAN e DIV.
1- SPAN: serve para adicionar, no meio da frase, algum estilo à uma(ou mais) palavra(s) especifica(s).
Essa é a tag de "Container genérico". Por padrão, SPAN é usada INLINE. EXEMPLO:
Lorem ipsum dolor, sit amet consectetur adipisicing elit.

2- DIV: Também é uma tag genérica. É um bloco. EXEMPLO:
